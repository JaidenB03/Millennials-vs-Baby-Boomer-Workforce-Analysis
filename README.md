# Millennials vs. Baby Boomers Workforce Analysis

## Project Overview

This project presents an interactive Tableau dashboard analyzing workforce trends between Millennials and Baby Boomers across U.S. metropolitan areas using EMSI labor market data from 2007–2013.

The dashboard explores employment growth, workforce participation, and regional job market changes to help identify labor market trends across generations.

---

## Business Problem

Organizations and workforce planners need to understand how employment trends differ between generations to support hiring strategies, workforce planning, and regional economic development.

This dashboard was created to answer questions such as:

- Which metropolitan areas experienced the greatest job growth?
- How did employment change between 2007 and 2013?
- Which cities experienced the highest workforce growth?
- How do Millennials and Baby Boomers compare across metropolitan areas?
